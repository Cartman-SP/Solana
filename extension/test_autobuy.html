Тестирование редиректа основного окна

Эта страница тестирует функциональность редиректа основного окна при получении autobuy=true

Инструкции по тестированию

  1. Откройте основное окно с trade.padre.gg
  2. Откройте боковую панель расширения
  3. Нажмите кнопку теста
  4. Проверьте, что основное окно редиректится на правильный URL
